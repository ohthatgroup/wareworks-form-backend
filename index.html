WareWorks Form Backend

Secure application processing system for WareWorks employment applications

Backend Services

This site provides backend functions for the WareWorks application form with secure document processing, I-9 compliance, and automated workflows.

Available Functions

  • Configuration Endpoint
    Provides secure configuration data for the application form
    Access Configuration
  • Form Submission Endpoint
    Handles application submissions with PDF generation and email notifications (POST only)
    Endpoint: /.netlify/functions/submit-application
  • Address Autocomplete
    Server-side proxy for Google Places autocomplete functionality
    Endpoint: /.netlify/functions/autocomplete-address

Features

Document Processing

Secure file upload handling with validation and storage

I-9 Compliance

Automated I-9 form generation and verification workflows

PDF Generation

Complete application packages with embedded forms

Email Notifications

Automated admin alerts with document attachments
